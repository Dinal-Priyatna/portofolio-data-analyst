# Analisis Data Menggunakan Microsoft Excel

## Deskripsi Proyek
Folder ini berisi analisis data yang dilakukan menggunakan Microsoft Excel untuk menghasilkan insight berbasis data. Fokus analisis adalah mengolah data mentah menjadi informasi yang siap digunakan untuk pengambilan keputusan.

## Tujuan Analisis
- Melakukan pembersihan dan pengolahan data (data cleaning)
- Menggunakan fungsi Excel untuk perhitungan (SUMIF, VLOOKUP, dsb.)
- Membuat PivotTable untuk eksplorasi pola data
- Visualisasi data menggunakan grafik

## Tools yang Digunakan
- Microsoft Excel
