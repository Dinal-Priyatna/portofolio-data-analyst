# Analisis Data Menggunakan Python

## Deskripsi Proyek
Folder ini berisi analisis data menggunakan Python dalam format Jupyter Notebook (.ipynb). Analisis dilakukan untuk eksplorasi data, visualisasi, serta penarikan insight secara terstruktur.

## Tujuan Analisis
- Melakukan data cleaning dan eksplorasi (EDA)
- Membuat visualisasi menggunakan library Python
- Menyusun insight berdasarkan hasil analisis

## Tools & Library yang Digunakan

- Jupyter Notebook
- Pandas, NumPy, Matplotlib (opsional sesuai kebutuhan)
