# Analisis Data Menggunakan SQL

## Deskripsi Proyek
Folder ini berisi query SQL yang digunakan untuk melakukan eksplorasi dan analisis data langsung dari database. Analisis difokuskan untuk menghasilkan insight yang relevan melalui query yang efisien dan tepat sasaran.

## Tujuan Analisis
- Mengambil dan memfilter data dari database
- Melakukan agregasi data (SUM, COUNT, GROUP BY, dsb.)
- Menjawab kebutuhan analisis atau pertanyaan bisnis berbasis data

## Teknologi yang Digunakan
- SQL (MySQL / PostgreSQL / Standard SQL — dapat disesuaikan)

## File dalam Folder Ini
- **query.sql** — berisi kumpulan query utama

## Status
Analisis ini merupakan bagian dari portofolio dan dapat diperluas dengan skenario analisis tambahan.
